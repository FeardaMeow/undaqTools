Installation
Tested on Windows 7 with Python 2.7 (32-bit and 64-bit)
Using the Unofficial Windows Binaries for Python Extension Packages provided by Christoph Gohlke:
  - h5py-2.1.2.win-amd64-py2.7.exe
  - numpy-MKL-1.7.0.win-amd64-py2.7.exe
  - scipy-0.11.0.win-amd64-py2.7.exe
  
 If h5py fails to import might need to reinstall Numpy and Scipy.
